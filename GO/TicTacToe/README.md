# Tic-Tac-Toe (Крестики-нолики)
Серверное приложение на Go, реализующее игру «Крестики-нолики» с поддержкой нескольких одновременных игр, разделением на слои и вычислением хода компьютера с помощью алгоритма **Minimax**.


## Описание проекта

Приложение предоставляет HTTP API для игры в «Крестики-нолики».  
Каждая игра имеет уникальный идентификатор (UUID) и собственное игровое поле.

Бизнес-логика игры изолирована от слоёв хранения данных, веб-интерфейса и внедрения зависимостей.  
Для управления зависимостями используется библиотека **uber/fx**.

## Используемые технологии

- Go
- net/http
- sync.Map
- uber/fx
- UUID
- Minimax algorithm

## Запуск

```
go run main.go
```
