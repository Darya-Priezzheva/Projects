# Concurrency in Go

Знакомство с механизмами конкурентности в Go. 
Работа с каналами для синхронизации горутин, а также обмен данными между ними. 
Изучение пакета sync.

---

## Секундомер асинхронных задач

Консольная программа на Go, демонстрирующая конкурентный запуск горутин и синхронизацию их выполнения.
Запускает N горутин, каждая из которых засыпает на случайное время до M миллисекунд, и дожидается их завершения.

После выполнения выводит список пар <номер горутины, время сна>, отсортированный по убыванию времени выполнения.
В задании используется пакет sync, запрещено применение каналов

## Запуск

```
cd stopwatch
```

```
go run ex.1 -n <количество горутин> -m <время в мс>
```

---

## Генератор квадратов числа

Программа состоит из генератора чисел и функции возведения в квадрат, запущенных конкурентно.
Числа последовательно передаются через каналы, обрабатываются и выводятся в стандартный поток вывода

## Запуск

```
cd generator
```

```
go run ex.2 -k <число> -n <число>
```

---

## Тикер

Асинхронный тикер, работающий с заданным шагом в секундах и выводящий информацию о каждом тике.
Программа корректно обрабатывает системные сигналы SIGINT и SIGTERM и завершает работу по их получению.


## Запуск

```
cd ticker
```

```
go run ex.3 -k <интервал в мс>
```

## LRU Cache с использованием Generics

Пакет на Go, реализующий потокобезопасный LRU-кэш с использованием generics.
Поддерживает операции Set, Get и Clear. Сложность O(1).

## Запуск

```
cd lrucache
```

```
go run ex.4
```
