# Основы синтаксиса и ввода-вывода

## Описание проекта
Небольшие программы на Go ля реализации консольных утилит

## Консольный калькулятор
Консольное приложение на Go для выполнения базовых арифметических операций с числами типа float64.
Поддерживает сложение, вычитание, умножение и деление с точностью до 3 знаков после запятой.

Реализована пошаговая обработка пользовательского ввода с подсказками и валидацией данных.
При вводе некорректных значений программа выводит сообщение об ошибке и запрашивает ввод повторно.
В проекте используется только стандартная библиотека Go.

### Запуск

```
cd Calculator
```

```
go run ex.1
```

## Наиболее часто встречающиеся слова

Программа для анализа строки и поиска K наиболее часто встречающихся слов.
Подсчитывает количество вхождений каждого слова, сортирует результаты по убыванию частоты, а при равной частоте — лексикографически.

Корректно обрабатывает граничные случаи: пустую строку и ситуацию, когда K больше количества уникальных слов.
Проект покрыт unit-тестами с использованием стандартного пакета testing.

### Запуск

```
cd MostFreqWords
```

```
go run ex.2
```

## Пересечение срезов
Консольная программа для поиска пересечения двух неупорядоченных списков целых чисел (int).
Результат возвращается в том порядке, в котором элементы встречаются в первом списке.

Предусмотрена обработка ошибок ввода и сообщений для случая пустого пересечения.
Проект демонстрирует работу со срезами, отображениями (map) и валидацией входных данных.

### Запуск

```
cd SliceIntersection
```

```
go run ex.2
```

## Журнал посещений
Интерактивное консольное приложение для ведения журнала посещений пациентов в медицинском учреждении.
Поддерживает команды сохранения визита, просмотра истории посещений пациента и получения последнего визита к определённому специалисту.

Данные хранятся только в памяти с использованием структур и map.
Реализована собственная ошибка PatientNotFoundError для обработки случаев отсутствия пациента в системе.
