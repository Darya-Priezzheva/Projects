# Projects (C / Go)

Этот репозиторий объединяет мои учебные проекты, выполненные в процессе обучения в School 21 — школе программирования, ориентированной на практику и peer-to-peer обучение.

В проектах я отрабатываю:
- системное программирование на C
- разработку сервисов и приложений на Go
- работу с архитектурой, слоями приложения и алгоритмами

Каждый проект представляет собой самостоятельное задание с чёткими требованиями и ограничениями.
