# C Library: s21_matrix

Статическая библиотека на языке C для работы с матрицами, реализующая основные операции и поддержку чисел с точностью до 6 знаков после запятой.

## Описание проекта

Библиотека `s21_matrix.a` предоставляет функции для создания, удаления и выполнения арифметических и линейных операций с матрицами.  
Проект разработан с использованием стандарта **C11** и ориентирован на **POSIX.1-2017**.

**Основные требования:**

- Работа с матрицей через структуру `matrix_t`
- Точность дробной части до 6 знаков
- Полное покрытие unit-тестами (не менее 80%) с использованием библиотеки **Check**


### Реализованные функции

- `create_matrix` — создание матрицы
- `remove_matrix` — очистка и уничтожение матрицы
- `eq_matrix` — сравнение матриц
- `sum_matrix` — сложение
- `sub_matrix` — вычитание
- `mult_matrix` — умножение матриц
- `mult_number` — умножение на число
- `transpose` — транспонирование
- `determinant` — вычисление определителя
- `calc_complements` — вычисление матрицы алгебраических дополнений
- `inverse_matrix` — поиск обратной матрицы

## Сборка и запуск

### Сборка библиотеки

```bash
make s21_matrix.a
```
Сборка и запуск тестов
```
make test
```
Генерация отчёта покрытия (gcov)
