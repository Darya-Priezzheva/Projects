# C Utilities: cat & grep

Разработка утилит Bash по работе с текстом: cat, grep.

## Описание проектов

### 1. `cat`
Утилита для вывода содержимого файлов в консоль с поддержкой флагов (поддерживается поведение, аналогичное GNU версии):
- `-b (GNU: --number-nonblank)` — нумерация непустых строк
- `-e (предполагает -v), -E (GNU only: без применения -v)` — отображение конца строк как `$`
- `-n (GNU: --number)` — нумерация всех строк
- `-s (GNU: --squeeze-blank)` — сжатие повторяющихся пустых строк
- `-t предполагает и -v (GNU: -T то же самое, но без применения -v)` — отображение табуляции как `^I`



### 2. `grep`
Утилита для поиска текста с использованием регулярных выражений с флагами:
- `-e` — указание шаблона поиска
- `-i` — игнорирование регистра
- `-v` — инвертирование совпадений
- `-c` — подсчет количества совпадений
- `-l` — вывод только имен файлов с совпадениями
- `-n` — нумерация строк с совпадениями
- `-h` — не выводить имена файлов
- `-s` — подавление ошибок о файлах
- `-f` — чтение шаблонов из файла
- `-o` — вывод только совпадающих частей

Реализована работа с файлами и поддержка регулярных выражений.

## Как собрать и запустить

1. Перейти в папку проекта:

   `cd C/cat`  или `cd C/grep`

2. Собрать проект с помощью Makefile:
   `make`

3. Запустить утилиту:

   `./s21_cat file.txt [флаги]`

   `./s21_grep pattern file.txt [флаги]`
