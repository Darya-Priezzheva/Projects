# C Library: s21_decimal

Библиотека на языке C для работы с типом `decimal` с поддержкой широкого диапазона значений и основных арифметических операций.

## Описание проекта

Проект реализует статическую библиотеку `s21_decimal.a` с набором функций для работы с десятичными числами.  
Библиотека разработана с использованием стандарта **C11** и ориентирована на **POSIX.1-2017**.

**Основные требования:**
- Структурное программирование
- Префикс `s21_` перед каждой функцией
- Использование двоичного представления числа с массивом `bits`
- Поддержка чисел в диапазоне:  
  `-79,228,162,514,264,337,593,543,950,335` … `+79,228,162,514,264,337,593,543,950,335`
- Отсутствие устаревших функций и типа `__int128`
- Полное покрытие unit-тестами (не менее 80%) с использованием библиотеки **Check**

### Функции библиотеки

Реализованы функции для:
- Арифметики (`add`, `sub`, `mul`, `div`, `mod`)
- Логических операций (`is_equal`, `is_greater`, `is_less`)
- Преобразований (`from_int_to_decimal`, `from_decimal_to_int`, `from_float_to_decimal`, `from_decimal_to_float`)
- Округления (`truncate`, `round`, `floor`, `negate`)  

## Сборка и запуск тестов

Сборка библиотеки

```bash
make s21_decimal.a
```
Сборка и запуск тестов
```bash
make test
```
Генерация отчёта покрытия (gcov)

```bash
make gcov_report
```
